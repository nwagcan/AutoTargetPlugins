AutoTargetControl (v28)
By NWAGCAN

A BetterDiscord plugin for automated voice moderation and self-protection.

° Features
Anti-Mute: Automatically unmutes you if a moderator mutes you.

Target Mute: Keeps a specific User ID server-muted.

Target Disconnect: Kicks a specific User ID from voice channels.

Optimized: Fixed "400 Bad Request" errors for faster performance.

° Quick Start
Install: Drop the .plugin.js file into your BetterDiscord plugins folder.

Open Menu: Press F4.

Configure: Paste the Target User ID, choose your Mode (Mute/Disconnect), and click Start.

Save: Click Save Settings to activate.

° Requirements
You must have Mute/Move Members permissions in the server.

Use responsibly according to Discord ToS.
